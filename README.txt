【部署到 GitHub Pages】
把這個資料夾的檔案上傳到 repo 根目錄（覆蓋舊檔），Settings → Pages 開啟即可。

【更新後按鈕沒反應？】
PWA 會快取舊版：
- Android：Chrome 清除該網站資料或刪除主畫面圖示後重裝
- iPhone：設定 → Safari → 進階 → 網站資料 → 刪掉該 github.io 網址

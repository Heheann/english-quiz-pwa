【安裝到手機（一定要 HTTPS）】
1) 把這個資料夾內容上傳到 GitHub repo 根目錄
2) Settings → Pages → Deploy from a branch → main + /(root)
3) 用手機開你的 https://xxxx.github.io/xxxx/ 網址
   - Android Chrome：⋮ → 安裝應用程式 / 加到主畫面
   - iPhone Safari：分享 → 加入主畫面

【更新後按鈕沒反應？】
PWA 會快取舊版：
- Android：Chrome 清除該網站資料或刪除主畫面圖示後重裝
- iPhone：設定 → Safari → 進階 → 網站資料 → 刪掉該 github.io 網址
